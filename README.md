基於 **FastAPI** 的 AI 股市分析 API，提供即時市場數據與智能分析服務。

## ✨ 主要特色

- 🤖 **AI 智慧分析**：採用 GPT-4.1 進行股票技術面與基本面分析
- 📊 **即時股價資料**：整合 Yahoo Finance 提供最新股價資訊
- 📰 **新聞資訊整合**：自動抓取相關股票新聞進行分析
- 💹 **籌碼面分析**：提供三大法人買賣超資料
- 🔍 **智慧股票查詢**：支援股票名稱自動轉換為代號


### 支援範圍
- 🏢 台灣上市、上櫃股票  
- 📈 台灣加權指數、櫃買指數 (^TWOII)
